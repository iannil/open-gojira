## 前端 Docker 构建修复

用户尝试 `docker compose build frontend` 时遇到 `npm run build` 退出码 2。

### 根因

TypeScript 编译错误：
1. **ReportsPage.tsx** (L187, L250): 使用了 `ResearchReportSummary` 类型但未从 `../../api/research` 导入 — 该类型确实存在且已导出，只是缺少 import 语句。
2. **UniversePage.tsx** (L1): 导入了 `useCallback` 但从未使用，`tsc -b` 的 `noUnusedLocals` 检查报错。

### 修复
- ReportsPage.tsx: 在 import 块中添加 `type ResearchReportSummary`
- UniversePage.tsx: 移除未使用的 `useCallback` 导入
