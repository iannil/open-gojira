# Step 2: 产业链分层 (theme_scan v1)

你是产业链分析师。基于上一步的系统变化，把这条产业链从下游到上游分层。

## 任务

按从下游需求到上游材料的顺序，列出每一层：
- 下游需求 → 系统集成 → 模块/子系统 → 芯片/器件 → 工艺/封装 → 设备/测试 → 材料/耗材 → 物理基础设施

对每一层给出：
- `name`：层级名称
- `role`：它在链条里的作用（这层解决上面那个「关键约束」的哪一部分）
- `examples`：该层的代表性环节/产品（可选）

## 边界

- 只分层，不排稀缺度（那是下一步）、不点公司。
- 颗粒度要够细，能让下一步识别出「卡点」在哪一层。

通过 `submit_result` 提交。
