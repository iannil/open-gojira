## 三、数据生命周期（完整链路）

```
数据源层                    存储层                    业务逻辑层                    展示层
─────────                  ──────                    ───────────                   ──────
Lixinger API ─────┐
(理杏仁)          │    ┌── SQLite ──┐
  - K线数据       ├──→ │ stocks     │ ──→ analysis_service ──→ 三步分析向导
  - 财务报表      │    │ analyses   │ ──→ valuation_service ──→ 估值仪表板
  - 估值指标      │    │ valuations │ ──→ financial_service ──→ 财务分析页
  - 分红历史      │    │ dividends  │ ──→ dividend_service  ──→ 分红仪表板
  - 行业数据      │    │ holdings   │ ──→ holding_service   ──→ 持仓管理
  - 资金流向      │    │ backtest_* │ ──→ backtest_service  ──→ 回测系统
CSV 导入 ─────────┘    │ screener_* │ ──→ screener_service ──→ 选股器
                       │ discipline │ ──→ discipline_service→ 交易纪律
                       └────────────┘
```

**完整用户旅程**：
1. **选股** → 选股器筛选 / 手动输入股票代码
2. **研究** → 三步分析向导（行业→商业模式→财务）+ 行业模板参考
3. **估值** → PE/PB 百分位 + 分红分析 + EPS 计算 + 综合估值
4. **深度分析** → 财务三表 + 同业对比 + DCF 计算 + 异常检测
5. **决策** → 买入前检查清单（8 项门控）
6. **执行** → 记录持仓 → 仓位监控
7. **管理** → 再平衡（未完成）→ 止盈提醒（未完成）
8. **复盘** → 投资日记 + 幻象自测 + 回测验证
