## 1. 全局布局与导航

页面顶部为**水平导航栏**，按功能分组：

| 分组 | 包含页面 |
|---|---|
| **入口** | 主看板 |
| **研究** | 研究报告 |
| **组合** | 持仓组合 · 股息红利 · 券商费率 |
| **数据** | 股票池 · 市场指数 · 估值分析 · 数据管理 |
| **执行** | 草稿 · 成交流水 |
| **自动化** | 定时任务 · 监控配置 · 公司行动 · 审计日志 |
| **质量** | Eval Set |

导航栏右侧显示系统版本号。点击任意导航项即可跳转到对应页面。

页面顶部有时会出现 **系统告警横幅**（红色/黄色），显示未解决的系统告警数量，点击可跳转到监控配置的告警中心查看详情。

## 2. 入口 → 主看板

**路径：** `/`  
**核心用途：** 一站式驾驶舱，展示系统核心状态和待办事项。

### 页面模块

| 模块 | 内容 |
|---|---|
| **系统状态** | 月度 LLM 成本（含硬熔断/软上限进度条）、观察池/候选池/持仓数量 |
| **组合评价** | 组合收益 vs 基准、超额收益、夏普比率、胜率、交易次数、Profit Factor、平均滑点 |
| **待办信号** | 待审批的买卖草稿列表（股票、方向、关卡、建议量、理由），以及信号告警 |
| **持仓概览** | 持仓数、组合市值、累计盈亏%、现金占比；持仓明细表（代码、数量、成本、现值、盈亏） |
| **候选漏斗** | 从 universe → watchlist → researched → candidate → holding → exited 六个阶段的股票数量 |
| **最近研究报告** | 最近生成的研究报告列表（股票、Pipeline 类型、评分、建议、证据等级） |
| **系统告警** | 未解决的系统告警列表（级别、类别、消息） |

### 可执行操作

- 点击"研究报告"按钮 → 跳转到研究报告页
- 点击"数据同步"按钮 → 跳转到数据管理页
- 点击草稿行的"审批"按钮 → 跳转到草稿页处理
- 点击股票代码 → 跳转到个股详情页
- 页面每 30 秒自动刷新看板数据

## 3. 研究 → 研究报告 + 个股详情

### 3.1 研究报告

**路径：** `/reports`  
**核心用途：** 浏览所有 AI 生成的研究报告。

#### 双栏布局

- **左栏（报告列表）：** 可按 Pipeline 类型（全部/深度研究/论文追踪/异动归因/财报精读/质量筛选）筛选，也可按股票代码搜索。
- **右栏（报告详情）：** 选中某条报告后展示完整内容。

#### 报告详情包含

- 股票代码（可点击跳转个股详情）
- 状态标签（运行中/冲突/过期/失败/拒绝）
- Pipeline 类型、建议、证据等级、评分
- **数据冲突告警**（LLM 值与数据库值不一致的字段及差异百分比）
- **红线否决告警**（触发的红线规则及处理动作）
- Markdown 报告正文
- 生成时间、过期时间、Prompt 版本

#### 可执行操作

- 筛选 Pipeline 类型、搜索股票代码
- 点击报告行查看详情
- 点击"刷新"按钮手动刷新

### 3.2 个股详情

**路径：** `/stock/:code`（如 `/stock/600519`）  
**核心用途：** 单只股票的深度研究与历史报告。

#### 触发研究面板

| 控件 | 说明 |
|---|---|
| Model Tier 选择器 | 选择 LLM 模型等级：sonnet（默认，~$0.028/次）、opus（最贵最优，~$0.11/次）、haiku（最便宜，~$0.001/次） |
| Web 搜索开关 | 推荐开启，关闭后 LLM 仅凭内部知识回答 |
| 触发研究按钮 | 正常触发（遵守 30 天缓存）或强制重跑（绕过缓存） |

每次研究触发共 6 次 LLM 调用（数据收集 + 4 位大师并行分析 + 综合合成）。

#### 报告展示

- **最新报告：** 展示最近生成的研究报告，生成中时自动轮询（每 3 秒）
- **历史研究：** 过往所有报告列表，每项显示时间、Pipeline、评分、建议、证据等级

## 4. 组合 → 持仓 + 股息 + 费率

### 4.1 持仓组合

**路径：** `/portfolio`  
**核心用途：** 当前持仓总览——市值、盈亏、权重分布。

#### 组合评价区

- 基准对比：组合收益 vs 基准收益、超额收益
- 夏普比率、胜率、交易次数、Profit Factor、平均滑点

#### 汇总指标

| 指标 | 说明 |
|---|---|
| 持仓数 | 当前持有标的数量 |
| 组合市值 | 当前总市值（含成本参考线） |
| 累计盈亏 | 总盈亏金额 + 盈亏百分比 + 年化收益率 |
| 现金占比 | 现金占总组合比例 + 现金储备金额 |

#### 持仓明细表

每行展示：股票（可点击跳转详情）、名称、行业、持仓量、成本价、现值、盈亏、盈亏%、年化%、权重%。

> ⚠️ 持仓数据**完全从 Trade 账本派生**，不可在此页面直接修改。如需调整，请通过成交流水页录入或红冲交易。

### 4.2 股息红利

**路径：** `/dividend`  
**核心用途：** 查看持仓股票的股息分红记录。

### 4.3 券商费率

**路径：** `/fee-configs`  
**核心用途：** 管理券商佣金、印花税、过户费等费率配置。

## 5. 数据 → 股票池 + 市场 + 估值 + 数据管理

### 5.1 股票池

**路径：** `/universe`  
**核心用途：** 浏览和管理个人股票池，以及全市场股票筛选。

#### 两种视图

| 视图 | 说明 |
|---|---|
| **我的 Universe** | 个人关注的股票列表，支持分层、安全主题、行业、选择权评分等多维度筛选 |
| **全市场** | 全部 A 股，支持 PE/PB/股息率等基本面筛选（服务端分页） |

#### 我的 Universe 表格列

| 列 | 说明 |
|---|---|
| 代码、名称 | 点击代码跳转个股详情 |
| 分层 | 核心（金）/ 卫星（蓝）/ 重点（绿）/ 未设（灰） |
| 能力圈 | 开关切换，即时保存 |
| 主题、行业、选择权 | 安全主题分类 |
| 预案状态 | 运行中/已暂停/已归档/无预案 |
| 持有状态 | 显示持有权重百分比 |
| PE%、DYR | 估值百分位和股息率 |

#### 可执行操作

- 切换"我的 Universe"和"全市场"视图
- 按代码/名称搜索
- 展开筛选面板设置多层过滤条件
- 切换股票的"能力圈"开关
- 点击代码跳转个股详情

### 5.2 市场指数

**路径：** `/market`  
**核心用途：** 查看主要市场指数行情。

### 5.3 估值分析

**路径：** `/valuation`  
**核心用途：** 查看股票的估值数据（PE、PB 等百分位）。

### 5.4 数据管理

**路径：** `/data-management`  
**核心用途：** 数据同步、健康监控、质量检查、清理的中央管理中心。

> "数据不准，下游全错"——这是所有策略和预案运行的输入基座。

#### 顶部横幅

显示全 A 股覆盖状态：覆盖股票数、有估值数据的股票数、覆盖率百分比。

- **全量覆盖模式**：每日自动同步全 A 股基础数据
- **手动模式**：仅同步手动添加的股票，可点击"启用全量覆盖"按钮触发 bootstrap Pipeline

#### 五个标签页

| 标签 | 功能 |
|---|---|
| **数据健康概览** | 各类数据的更新新鲜度（估值/财报/K线/分红）、API 用量、死信队列统计 |
| **数据同步** | Pipeline 执行历史列表，可启动/重试/取消 Pipeline 运行 |
| **股票池管理** | 添加或移除股票池中的股票 |
| **数据质量** | 各类数据的质量评分和改进建议 |
| **数据清理** | 按数据类型和时间范围清理过期数据 |

## 6. 执行 → 草稿 + 成交流水

### 6.1 草稿

**路径：** `/drafts`  
**核心用途：** AI 自动生成的买卖草稿队列——审批、执行或取消的入口。

#### 统计行

- 待处理 / 买入 / 卖出 三类草稿的数量

#### 筛选标签

| 标签 | 说明 |
|---|---|
| **待处理** | 等待审批的草稿 |
| **已执行** | 已成交的历史草稿 |
| **已取消** | 已取消的草稿 |

#### 草稿卡片内容

每张草稿卡片展示：

- **股票代码** + **方向**（买入/卖出标签）
- **策略等级**（积极/稳健）
- **论点状态**（健康/证伪）
- **来源**（规则引擎/质量筛选/深度研究/论点跟踪/主题扫描/手动）
- **触发时间** + **过期倒计时**（<24h 红色提示）
- **建议数量** 或 **建议加仓/减仓百分比**
- **目标价** 和 **仓位逻辑说明**
- **理由**（完整文本）

#### 可执行操作

- **成交：** 点击"成交"按钮，输入实际成交价、数量和成交时间
- **取消：** 点击"取消"按钮放弃该草稿
- 卖出草稿会显示 T+1 可用股数提示

### 6.2 成交流水

**路径：** `/trades`  
**核心用途：** 交易账本——所有已成交买卖、分红、公司行为的完整记录。

#### 现金余额

页面顶部显示当前现金余额及更新时间。若未初始化显示提示。

#### 筛选栏

- **按代码搜索**：输入股票代码过滤
- **按方向过滤**：买入/卖出/分红/公司行为

#### 交易列表

每行展示：ID、股票代码、方向（颜色标签）、成交价、数量、成交时间、成交额、费用（佣金+印花税+过户费）、来源。

#### 可执行操作

| 操作 | 说明 |
|---|---|
| **录入成交** | 手动记录一笔交易（股票、方向、价格、数量、时间、佣金覆盖） |
| **红冲** | 撤销一笔已成交的交易（生成反向交易，现金账户同步回滚） |
| **入金/取现** | 记录资金的存入或取出 |

## 7. 自动化 → 定时任务 + 监控 + 公司行动 + 审计日志

### 7.1 定时任务

**路径：** `/scheduler`  
**核心用途：** 后台 cron 调度管理——查看、编辑、启停、手动触发定时任务。

#### 任务管理标签

表格列出所有注册的定时任务：

| 列 | 说明 |
|---|---|
| 任务 ID | 唯一标识 |
| 描述 | 任务功能说明 |
| 调度规则 | cron 表达式（可点击直接编辑）+ 中文可读描述 |
| 状态 | 开关切换 |
| 上次执行 | 执行时间 + 状态（成功/失败）+ 耗时 |
| 下次执行 | 预计下次执行时间 |
| 操作 | 手动触发按钮（Popconfirm 确认） |

#### 执行日志标签

最近 100 条执行记录，可按任务 ID 筛选，展示：任务、状态、开始时间、耗时、结果摘要、错误信息。

### 7.2 监控配置

**路径：** `/monitoring`  
**核心用途：** 止损止盈规则 + 通知通道 + 告警中心的统一管理。

> 规则由 scheduler 每 5 分钟评估；告警按级别过滤分发到通道。

#### 四个标签页

| 标签 | 功能 |
|---|---|
| **运营度量** | Pipeline 运营健康（成功率、明细）、LLM 费用 & Token 消耗、日级别费用趋势折线图 |
| **通知通道** | 管理通知渠道（站内/微信/邮件/钉钉/Telegram），可新增、编辑、测试、启用/禁用、删除 |
| **止损止盈规则** | 为持仓股票设置止损（成本百分比/追踪止损/固定价格）和止盈规则，支持开关和删除 |
| **告警中心** | 系统告警历史，可按级别/类别筛选，支持单条解决和批量解决 |

### 7.3 公司行动

**路径：** `/corp-actions`  
**核心用途：** 查看公司行动记录（分红、送转、配股等）。

### 7.4 审计日志

**路径：** `/audit-log`  
**核心用途：** 系统操作审计溯源。

## 9. 常见问题

### 数据不更新怎么办？

1. 前往 **数据管理** → **数据同步** 标签页，查看 Pipeline 运行状态
2. 如有失败任务，点击重试
3. 检查 **定时任务** 中相关数据同步 job 是否启用

### 持仓数据不对？

持仓完全从 Trade 账本派生。如数据异常：
1. 前往 **成交流水** 检查交易记录是否完整
2. 如有错误交易，使用"红冲"功能撤销后重新录入

### 研究报告为空？

1. 确认股票池中已包含该股票（前往 **股票池** 页面）
2. 前往 **个股详情** 页，点击"触发研究"按钮
3. 查看研究报告页面筛选正确的 Pipeline 类型

### LLM 调用失败或成本过高？

1. 前往 **监控配置** → **运营度量** 查看 LLM 费用和调用统计
2. 在个股触发研究时选择更便宜的模型（haiku）
3. 系统有 $150/月硬熔断保护

### 系统告警频繁？

前往 **监控配置** → **告警中心** 查看和处理未解决的告警，也可在 **通知通道** 配置告警分发规则。
